# timetracker
This android app allows the user to track how much time was spent on work done for one of several projects. 
So assume that you work for three projects (A, B, C) and you have a meeting at 8:00 for 1 hour related to project A.
You would press a button A, that starts a timer and at the end of the meeting you press the button again to stop it 
because afterwards you start to work on project B (preparing some tickets for example). After you stopped A and then started 
B the B-timer starts and you stop it when your'e done with the preparation of tickets for project B. Maybe you work 
for A again so you start the A-timer and so on. Instead of starting and stopping, you could also start, say timer C, 
when you start doing a call related to project C, which will stop the currently running timer. 

All the activities are logged on the phone. In a future release the app will also be able to sync against a server, that will 
be part of this project.
